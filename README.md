Linux System Monitoring Dashboard using AWS EC2

Project Overview

This project demonstrates Linux system monitoring using AWS EC2, Ubuntu Linux, htop, and Glances.

Tools Used

- AWS EC2
- Ubuntu Linux
- htop
- Glances

Features

- CPU monitoring
- Memory monitoring
- Process monitoring
- Real-time Linux monitoring

Commands Used

sudo apt update
sudo apt install htop -y
sudo apt install glances -y
htop
glances

Outcome

Successfully monitored Linux server performance and system resources using terminal-based monitoring tools.
